## 📽️ The Reading Experience: Custom Cinematic Engine

To mirror the emotional beats of the writing, the stories leverage a bespoke web engine (`story-cinematic.js` & `story-cinematic.css`) that transforms a simple reading layout into a multisensory experience:

* **Emotional Theme Transitions:** As you scroll, the screen seamlessly shifts colors, overlays, and color palettes (`data-chapter-theme`) to transition the visual mood from a warm sunset to midnight grief, cosmic isolation, or heavy tension.
* **Word-by-Word Reveal:** Key sentences and deep thoughts fade in word-by-word synchronized with the reader's scroll pace, creating a deliberate reading tempo.
* **Immersive Focus Mode:** Readers can toggle a distraction-free mode that auto-hides header controls and sidebar navs, letting the text and background textures fill the screen.
* **Tactile 3D Shelving:** The landing page (`index.html`) displays the stories on an interactive 3D bookshelf with dynamic card tilt, shimmer, and hover overlays.
* **Cinematic Splash Screen:** A beautiful entry screen that simulates loading, displaying custom branding ("WR Presents") and a progress tracker. Clicking the gold-rimmed "ENTER ARCHIVE" button crossfades smoothly into the bookshelf.

## 🛠️ How the Platform is Built

The portfolio is lightweight, responsive, and runs entirely client-side using standard web standards:
* **Prose Structure:** Semantic HTML5 elements designed for high readability and structured layouts.
* **Styling & Motion:** Vanilla CSS featuring custom variables for multi-emotional themes, combined with fluid responsive typography (`clamp`).
* **Libraries (via CDN):**
  * **Lenis:** For inertial, buttery-smooth scrolling.
  * **GSAP & ScrollTrigger:** Drives the paragraph fades, word reveals, and chapter entrance animations.
* **Effects:** HTML5 Canvas particles and custom SVG noise filters for realistic film grain.
